## Features

- /timerstart Command (Example /timerstart &4&lMaintenance 5h20m5s or /timer start <placeholder> 1d5h) - Adaptative placeholder
- Timing simple
- Clean code

## Permissions
  
- /timerstart > BungeePerm: bungeecord.timerstart
